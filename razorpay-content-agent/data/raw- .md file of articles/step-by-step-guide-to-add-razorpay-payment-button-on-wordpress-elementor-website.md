## How to install Razorpay Payment Button plugin?

You can add the plugin directly from your WordPress account following the instructions below, or [download the WordPress plugin as a .zip file](https://wordpress.org/plugins/razorpay-payment-button/) or [download the Elementor plugin as a .zip file](https://wordpress.org/plugins/razorpay-payment-button-elementor/) and upload it to your WordPress admin dashboard.

#### Install Razorpay button plugin from within WordPress:

*   Log in to your WordPress website admin dashboard
*   In the left sidebar menu, navigate to **Plugins**>**Add New**
*   Search for _the_[_Razorpay Payment Button for WordPress plugin_](https://wordpress.org/plugins/razorpay-payment-button/)_(For WordPress site page),_[_Razorpay Payment Button for Elementor_](https://wordpress.org/plugins/razorpay-payment-button-elementor/)_(For an Elementor page)_
*   In the upper right, click **Install Now**

*   Click **Activate Plugin**

### Follow these steps after the activation flow:

*   Log in to your Razorpay Account
*   In the left sidebar menu, navigate to **Settings → API Keys → Copy API Keys**

Step 2
